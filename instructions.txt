PascalCase -> messages
snake_case -> field/attributes
UPPER_CASE -> enum values
camelCase -> func names

protobuf files compilation:
In wsl :
 activate the myenv environment -> chmod +x scripts/proto.sh -> ./scripts/proto.sh

python -m grpc_tools.protoc ^ -I . ^ --python_out=. ^ --grpc_python_out=. ^ proto\common.proto ^ proto\namenode.proto ^ proto\datanode.proto

for datanode: python -m datanode.app.main
for namenode: python -m namenode.app.main

To run proto files:
  In Git Bash:
    cd storec
     ./scripts/proto.sh


TO run namenode :
  In Git Bash:
    cd storec
    ./scripts/init.sh

To run docker-compose.yaml( to run datanode in docker container ):

  cd storec

  docker compose up --build -d

  docker logs -f datanode-v3   # to see the logs in terminal instead of seeing in docekr desktop
  docker compose down    # To stop or remove the container 
  docker compose restart datanode  # To restart the DataNode container
